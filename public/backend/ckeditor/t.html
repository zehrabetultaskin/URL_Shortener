script file दो प्रकार के होते हैं | script file दो प्रकार के होते हैं |  internal file script और external file script

1. internal file script :

इसे HTML Document में script tag के बीच में लिखा जाता है

<script type="text/javascript"> //your code </script>

2.external file script:

इसे एक Seprate File में लिखा जाता है । File का extension .js होता है। .js small word में होगा | File का नाम कुछ भी रख सकते हैं | जैसे की myfirstfile.js और उस File को HTML Document में insert या call किया जाता है

<script type="text/javascript" src="myfile.js"> </script>

document.write("Hello, Teach Coders, This is My first Javascript")
